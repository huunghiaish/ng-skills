# ng:enhance-prompt

> **Forked from:** [google-labs-code/stitch-skills/skills/enhance-prompt](https://github.com/google-labs-code/stitch-skills/tree/main/skills/enhance-prompt)

Transforms vague UI ideas into polished, optimized prompts. Enhances specificity, adds UI/UX keywords, injects design system context, and structures output for better generation results.

## Install

```bash
git clone https://github.com/huunghiaish/ng-skills.git /tmp/ng-skills
cp -r /tmp/ng-skills/enhance-prompt/ ~/.claude/skills/
rm -rf /tmp/ng-skills
```

## Usage

```bash
/ng:enhance-prompt
```
