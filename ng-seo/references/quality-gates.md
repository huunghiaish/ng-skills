# Content Quality Gates

## Word Count Minimums
| Page Type | Min Words | Unique % | Notes |
|-----------|-----------|----------|-------|
| Homepage | 500 | 100% | Clear value proposition |
| Service/Feature | 800 | 100% | Detailed offering |
| Blog Post | 1,500 | 100% | In-depth content |
| Product | 400 | 80%+ | Unique descriptions, specs |
| Category | 400 | 100% | Unique intro, not just listings |
| Location (Primary) | 600 | 60%+ | Main service area |
| Location (Secondary) | 500 | 40%+ | Satellite locations |
| Landing Page | 600 | 100% | Focused conversion content |
| About | 400 | 100% | Company story, team |
| FAQ | 800 | 100% | Comprehensive Q&A |

## Location Page Thresholds
- **30+ pages:** ⚠️ WARNING — enforce 60%+ unique content, local info, testimonials
- **50+ pages:** 🛑 HARD STOP — require justification, legitimate presence proof
- Doorway page penalty risk: only city name changed, no local signals

## Title Tag
| Aspect | Requirement |
|--------|-------------|
| Length | 30-60 characters (Google truncates ~60) |
| Keyword | Near the beginning |
| Brand | At end (if included) |
| Uniqueness | Each page must be unique |

## Meta Description
| Aspect | Requirement |
|--------|-------------|
| Length | 120-160 characters |
| CTA | Include compelling call-to-action |
| Keyword | Include naturally |
| Uniqueness | Each page must be unique |

## Image Alt Text
| Aspect | Requirement |
|--------|-------------|
| Required on | All non-decorative images |
| Length | 10-125 characters |
| Content | Describe image, not "image.jpg" |
| Keywords | Natural, not stuffed |
| Decorative | Use `alt=""` or `role="presentation"` |

## Internal Linking
| Page Type | Target Links |
|-----------|-------------|
| Blog (1,500+ words) | 5-10 |
| Service page | 3-5 |
| Category | Links to all children |
| Product | 2-4 |

Anchor text: descriptive (not "click here"), varied, relevant.

## Content Freshness
| Type | Update Frequency |
|------|------------------|
| News | Hours/days |
| Blog (evergreen) | Annually |
| Product | When specs change |
| Service | Quarterly |

Required: publication date visible, last updated if revised.

## Safe vs Risky Programmatic Pages
**Safe:** Integration pages, template/tool pages, glossary (200+ words), product pages, user profiles.
**Risky:** Location pages (city-swapped), "Best X for Y" (thin), competitor alternatives (no real data), mass AI content.
